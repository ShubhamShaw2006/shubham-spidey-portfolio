# Shubham Shaw's Comic-Accurate Spider-Man Portfolio 🕷️

Welcome to my personal developer portfolio! This project leverages an incredibly unique and bespoke combination of raw CSS and procedural HTML5 Canvas mathematics to render a highly interactive, fluid, comic-book aesthetic inspired by Spider-Man.

## Features

- **Procedural Drag-to-Shoot Web Physics**: Clicking and dragging your mouse literally shoots a comic-accurate spider web featuring gravitational droop and thick tangled strands!
- **Interactive THWIP! Responses**: Everywhere you click natively spawns a randomized, dynamic comic-book action text bubble (e.g. WHACK, THWIP, ZAP).
- **Custom SVG Reticle Cursor**: Fully integrated default red Spidey web-shooter crosshair, turning blue when interacting with links.
- **Classic Print Halftones**: CSS-based radial gradient overlays to give genuine vintage comic book polka-dot print vibes!
- **GSAP Snappy Entrances**: Content boxes fly onto the screen utilizing advanced scroll triggers with snappy bouncing easing algorithms.
- **Fully Responsive**: Flawlessly adapts to mobile viewports including complex native touch integration for the web-shooter physics.

## Files Required to Run

This robust project is deliberately kept lightweight, relying on front-end browser languages without needing heavy node modules. Below is the list of crucial files required to run it:

1. \`index.html\` - The core structural layout of all segments (Hero, Experience, Projects, etc.).
2. \`assets/css/style.css\` - Features custom comic panel styling, SVG Cursors, Halftone overlays, and grid alignments.
3. \`assets/js/script.js\` - The engine! Handles the Drag-and-Drop canvas rendering, GSAP integrations, and mobile touch mappings.
4. \`assets/img/author_profile.jpeg\` - Your main central Hero Graphic portrait.
5. \`assets/img/portfolio_background.png\` - The cheerful daytime NYC Skyline illustration driving the background aesthetics.
6. \`assets/docs/ShubhamShaw_resume_26.pdf\` - Centralized directory for downloadable CV documents.

## How to Run

Because this project utilizes zero compile frameworks, getting started is practically instantaneous:

1. Clone or download all the files listed above into a single directory.
2. Open \`index.html\` directly in your browser. 
3. *Recommendation*: Using an HTTP server (such as Python's \`python -m http.server\` or VS Code's **Live Server** extension) ensures the font files and GSAP CDN scripts load with ideal permissions!
